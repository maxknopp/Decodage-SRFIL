Décodage SRFIL

Outil de décodage des log SRFIL

Cet outil permet de décoder un log SRFIL exécuté en mode debug.

Pour décoder un log, vous pouvez soit envoyer un fichier texte contenant les log via le bouton ci-dessous, soit copier/coller le résultat de la commande dans le formulaire plus bas.

Pour utiliser cet outil vous devez utiliser soit IE version 8 minimum soit Firefox version 3 minimum ou Chrome. Javascript doit également être activé.
